# Проект Место на Реакте с авторизацией и регистрацией

 Это интерактивный сайт, на котором можно добавлять фотографии, удалять их и ставить лайки. 

## Реализованные методы 

### Запросы к API:
- проверка jwt токена,
- авторизация с использованием localStorage,
- регистрация,
- получение данных профиля (user, cards),
- изменение данных профиля (avatar, name, about),
- добавление / удаление собственной карточки,
- постановка / снятие лайка.
### Локальное хранилище:
- при монтировании App описан эффект, проверяющий наличие токена и его валидности,
- при вызове обработчика onSignOut происходит удаление jwt,
- при вызове обработчика onLogin происходит сохранение jwt.
### Роутинг:
- роут / защищён HOC-компонентом ProtectedRoute,
- при попытке перейти по роуту / происходит редирект на /signin.
- после успешной авторизации происходит редирект на /.
### Модальные окна:
- изменение данных профиля,
- добавление карточки,
- просмотр фото при клике на карточку,
- предупреждение перед удалением карточки,
- уведомление об (не)успешной регистрации.
Модальные окна открываются при нажатии на соответствующий элемент интерфейса.
Закрываются при нажатии на иконку закрытия и на Escape.
### Общая информация:
- Разметка портирована в JSX,
- В работе использованы хуки (useState, useEffect, useContext, useHistory),
- используются компоненты <Switch />, <Route />, <Redirect />,
- роут / защищён HOC-компонентом ProtectedRoute,

Все запросы на авторизацию, регистрацию и проверку токена должны работают через сервис `https://auth.nomoreparties.co`. Запросы, на изменение данных профиля, доавление/удаление/лайк карточки работают с `https://mesto.nomoreparties.co/v1/cohort-16/`.
